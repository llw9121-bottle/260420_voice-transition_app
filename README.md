# 语音实时转录系统 🎙️

> 基于阿里云 AI 的桌面语音识别应用，帮你把录音快速转成结构化文档。

## 🤔 这是什么？能帮我做什么？

这是一个**桌面应用程序**，可以：

- 🎤 打开应用 → 点击开始录音 → 对着麦克风说话 → 实时转成文字
- ✨ 自动整理文本，可以去除语气词、分成自然段落
- 🔍 **独特功能**：自定义"关键行为匹配"，让 AI 帮你分析对话中出现了哪些特定行为（比如会议中的需求提出、问题反馈、决策安排），并自动生成统计表格

**适用场景：**

| 场景 | 用途 |
|------|------|
| **访谈采访** | 实时转录访谈内容，节省后整理时间 |
| **会议记录** | 自动记录会议内容，识别需求、问题、决策 |
| **心理咨询** | 记录咨询对话，识别情绪表达和问题 |
| **讲座听课** | 实时转写讲师内容，不错过重点 |
| **语音备忘录** | 想到什么直接说，自动转成文字笔记 |

## ✨ 主要优势

相比其他语音转文字工具，这个应用有什么优势：

1. **🎯 行为分析功能独有** - 你可以定义自己想要识别的行为类型（比如"用户提出需求"、"设计师反馈问题"），AI 会自动在文本中标注，并统计出现频率，生成汇总表格

2. **🖥️ 原生桌面应用** - 打开就能用，不需要浏览器，界面响应流畅

3. **🧑‍💻 新手友好** - 首次启动自动弹出配置向导，直接在界面输入 API Key 就能用，不用手动编辑配置文件

4. **🔒 数据隐私** - 音频通过阿里云 API 处理，你的 API Key 保存在本地电脑，不会上传到其他第三方服务器

5. **💸 按 token 付费** - 直接使用阿里云的付费服务，你自己掌控成本，没有中间商加价

6. **📊 多种导出格式** - 支持 Markdown、Word、JSON 三种格式，满足不同需求

## 🎯 功能亮点

### 核心功能
- **✅ 实时流式识别** - 边录音边识别，结果即时显示，不用等录音结束
- **✅ 暂停/继续录音** - 长时间录音中间需要休息？随时暂停，继续录制同一个文档
- **✅ 自动防崩溃保护** - 录音过程自动保存临时文件，意外重启可以恢复未完成录音
- **✅ 网络自动重连** - 网络临时断开后自动恢复，不丢失已转录内容
- **✅ 内存优化** - 长时间录音内存使用稳定，不会越用越卡

### 文本处理
- **四种格式化风格可选：**
  1. `raw` - 原始转录，原汁原味，不做处理
  2. `cleaned` - 基础清洗，去掉"嗯啊"语气词和重复内容
  3. `paragraphs` - 整理为自然段落
     - 默认快速模式：基于时间戳分割，不消耗 AI 额度
     - 可选 LLM 增强：让 AI 按语义重新分段，段落结构更自然
  4. `behavior_match` - **关键行为匹配**（特色功能）
     - 你定义要识别的行为 → AI 自动识别标注 → 末尾生成统计表格

- **逐步精细化处理**：清洗文本 → LLM语义段落整理 → 行为匹配，三步提升精度
- **超长文本支持**：自动分块处理，理论支持任意长度录音，不超限
- **多语言识别**：支持中文、粤语、英文、日语、韩语、德语、法语、西班牙语、俄语

### 行为匹配特色
- 自定义行为：支持添加 4-7 个你感兴趣的行为，每个行为可以写描述
- 内置常用模板：会议分析、求职面试、心理咨询、需求沟通、领导力评估，一键加载
- 配置导入/导出：备份你的配置，在多设备间同步
- 中英双语自动适配：根据识别语言自动选择对应提示词，效果更好

### 用户体验
- 实时录音时长显示，掌握录音进度
- 录音音量可视化，直观判断麦克风输入是否正常
- 红色闪烁录音状态指示灯，一眼看出是否正在录音
- 文本搜索功能，关键词高亮，快速定位内容
- 滚动锁定：锁定自动滚动，方便录音过程中查看历史内容
- 主题切换：支持 System / Light / Dark 三种主题，记住你的选择
- 快捷键支持：空格开始/暂停、Esc 停止、Ctrl+S 导出、Ctrl+B 行为配置
- 默认导出路径记忆：记住你上次选择的导出目录，不用每次都选

### 导出
- 三种格式：Markdown（适合分享）、Word（适合编辑排版）、JSON（适合二次开发）
- 行为匹配结果完整保留：原位标记 + 末尾统计表格

## 📖 使用指南

### 基本流程

| 步骤 | 操作 |
|------|------|
| 1 | 选择识别语言（默认中文） |
| 2 | 调整 VAD 静音检测时长：<br>- 对话访谈：400ms（默认）<br>- 朗读演讲：1000ms+ |
| 3 | 选择格式化风格（见下文说明） |
| 4 | （可选）如果用行为匹配，点击「行为配置」设置你要识别的行为 |
| 5 | 点击「开始录音」，允许麦克风访问 |
| 6 | 开始说话，实时看转录结果 |
| 7 | 点击「停止录音」结束转录 |
| 8 | 点击「导出」，选择格式保存文档 |

### 快捷键

| 快捷键 | 功能 |
|--------|------|
| `空格` | 智能切换：未开始→开始，录音中→暂停，暂停中→恢复 |
| `Esc` | 直接停止录音 |
| `Ctrl+S` | 导出文档 |
| `Ctrl+B` | 打开行为配置 |

### 格式化风格选择指南

| 风格 | 适用场景 | 是否消耗 AI Token |
|--------|---------|------------------|
| 原始(raw) | 需要原汁原味，不做处理 | ❌ 否 |
| 清洗(cleaned) | 只需要去掉语气词重复 | ❌ 否 |
| 分段(paragraphs) | 需要整理成段落 | ❌ 否（默认）/ ✅ 是（开启LLM增强） |
| 行为匹配(behavior_match) | 需要识别统计特定行为 | ✅ 是 |

### 行为匹配功能怎么用？

行为匹配是本应用的特色功能，可以让 AI 识别对话中你关心的特定行为。举个例子：

**会议分析场景：** 你定义以下四个行为：
1. **需求提出** - 提出新的需求或需求变更
2. **问题反馈** - 反馈遇到的问题或bug
3. **风险识别** - 指出潜在风险和问题
4. **决策安排** - 做出的决策和行动计划

录音转录完成后，AI 会：
- 在原文中标注哪里出现了哪个行为，给出置信度
- 在文档末尾自动生成统计表格，统计每个行为出现多少次，按置信度分类

**使用步骤：**
1. 格式化风格选择 `behavior_match`
2. 点击「行为配置」按钮
3. 可以选择：
   - 点击「加载模板」选择预置模板一键载入
   - 或者自己手动添加行为名称和描述
4. 保存配置，开始录音
5. 停止后等待 AI 分析完成
6. 导出就能看到带标注和统计表的文档

## ⚠️ 注意事项

### API Key 相关
- 🔑 **API Key 保存在你本地电脑**，不会上传到其他地方
- 如果需要修改 API Key，主界面右侧设置面板底部有「🔑 设置 API Key」按钮，随时可以改
- 不要把 `.env` 文件提交到公共仓库，已经在 `.gitignore` 中保护你

### 麦克风权限
- 第一次使用需要允许应用访问麦克风
- Windows：系统会弹出权限请求，点击允许即可
- macOS：系统设置 → 隐私与安全性 → 麦克风，允许应用访问

### 网络
- 需要稳定网络连接，因为识别是通过阿里云 API 实时进行的
- 如果网络临时断开，应用会自动尝试重连，一般 3 次重连失败才会停止
- 重连成功后会继续识别，已转录内容不会丢失

### 费用
- 费用由阿里云直接收取，本应用不收取任何费用
- 开通 DashScope 后可以在阿里云控制台查看消费明细
- 百炼大模型调用会额外消耗 token，价格参考阿里云官方定价

## 🐛 常见问题

**Q: 双击 exe 没反应？**

A: 请确保完整解压，不要直接在压缩包内双击运行，必须解压到文件夹再运行。

**Q: Windows 提示"已保护你的电脑"？**

A: 这是 Windows SmartScreen 对未签名应用的正常提示，点击「更多信息」→「仍运行」即可启动。

**Q: macOS 提示"无法打开，开发者无法验证"？**

A: 这是 macOS 对未签名应用的安全机制。解决方法：**右键点击图标 → 选择「打开」**，在弹出框点击「打开」。第一次这样操作后，以后就可以直接双击打开了。

**Q: 提示"网络连接失败"？**

A: 检查：
1. 网络是否能正常上网
2. API Key 是否正确输入
3. 阿里云账号是否有余额

**Q: 识别出来文字分段不对/断句太碎？**

A: 调整 VAD 静音检测时长：
- 如果断句太碎，调大 VAD 时长（比如从 400ms 调到 800ms 或 1000ms）
- 如果一句话太长没断开，调小 VAD 时长

**Q: 行为匹配结果不准确怎么办？**

A: 几个建议：
1. 行为描述写清楚一些，给 AI 更多信息
2. 行为数量控制在 4-7 个，不要太多
3. 确保启用了"LLM 段落整理"，先整理段落再匹配精度更高
4. 如果文本很长，确保开启了"自动分块"，避免超长文本上下文稀释
